# Weather Detector

A simple web application that fetches and displays real-time weather information based on user-input locations.

## Features
- Real-time weather data from an API.
- Dynamic background changes based on location.
- Responsive design.

## Installation
1. Clone the repository: `git clone <repository-url>`
2. Open `index.html` in a web browser.

## Usage
Enter a location in the search box to get the current weather information.
